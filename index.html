მაგიურ აკადემიაში

აქამდე თქვენ უბრალოდ აგროვებდით მონაცემებს — აწყობდით ინვენტარს. მაგრამ ინვენტარს აზრი არ აქვს, თუ მისი გამოყენება არ იცით.

„თუ ცვლადები არის ნივთები ჯადოსნურ ჩანთაში, ფუნქციები არის შელოცვები. ერთხელ ისწავლით შელოცვას და შემდეგ იყენებთ იმდენჯერ, რამდენჯერაც დაგჭირდებათ.“

გადაფურცლე, რომ შეისწავლო პირველი შელოცვა.

1. პირველი შელოცვა

შელოცვის გამოცხადება (Declaration) და გამოყენება.

// შელოცვის შექმნა
function castSpell(spellName) {
  let result = "Casted: " + spellName;
  return result; 
}

// შელოცვის გამოყენება
let attack = castSpell("Expelliarmus");
console.log(attack);

Console.log ილუზიაა, ნაპერწკალი ეკრანზე. Return არის ნამდვილი მაგია, რომელიც აბრუნებს რეალურ ძალას!

2. აურა (Scope)

ლოკალური მაგია მხოლოდ ოთახში მუშაობს, გლობალური კი ყველგან. 

let hogwartsMana = 1000; 

function healingRoom() {
  let roomMana = 200; 
  let hogwartsMana = 5000; // Shadowing
  console.log(hogwartsMana); // 5000
}

healingRoom();
console.log(hogwartsMana); // ისევ 1000
// console.log(roomMana); -> ERROR!

3. ჯოხის ქნევის ტექნიკა

Direct Execution (პირდაპირი)

castSpell() — ფრჩხილები ნიშნავს: "ახლავე ისროლე!"

Indirect Execution (ირიბი)

castSpell — ფრჩხილების გარეშე ნიშნავს: "ჯოხი მოვიმარჯვე და ველოდები ბრძანებას".

ამას მომავალში Event Listeners-ზე გამოვიყენებთ (მაგ: ღილაკზე დაჭერისას).

4. უსიტყვო მაგია (ES6)

უფროსკურსელები აღარ ყვირიან შელოცვებს. მათ გააქრეს სიტყვა `function`, ფრჩხილები და `return`. 

// ტრადიციული
function lumos() {
  return "Wand is glowing!";
}

// Arrow Function (თანამედროვე)
const patronus = (animal) => {
  return "Expecto: " + animal;
};

console.log(patronus("Stag"));

⚔️ დუელის კლუბი (მისია)

გახსენით Console (F12) და დაწერეთ:

  1. ფუნქცია calculateDamage(base, crit), რომელიც დააბრუნებს ნამრავლს.
  2. Arrow Function: healPotion(hp), რომელიც hp-ს დაუმატებს 50-ს.
  3. let myHealth = 100;
  4. გამოიძახეთ დარტყმა: myHealth = myHealth - calculateDamage(20, 2);
  5. აღიდგინეთ სიცოცხლე: myHealth = healPotion(myHealth);

გადაუღეთ სქრინშოთი კონსოლს!
